# ✈️ Flight Tracking App

Flight Tracking App is a mobile application built with React Native and Expo, offering a clean and modern interface for searching flights and viewing ticket details. Users can quickly search for flights by selecting departure–arrival locations, date, and passenger information, browse the results, and view detailed information for their booked tickets.

## Features
- Search flights by departure, arrival, date, and passengers
- View flight results
- Complete a ticket purchase
- See active and past tickets
- Boarding pass view with QR and flight details
- Clean, minimal, mobile-friendly interface

## Tech Stack
- React Native
- Expo
- TypeScript
- Async Storage

## Get started

### 1. Clone the project
   
- HTTPS
```bash
  git clone https://github.com/orcunakkaya/react-native-flight-tracking.git
   ```
- SSH
```bash
  git clone git@github.com:orcunakkaya/react-native-flight-tracking.git
   ```

### 2. Go to the project directory

```bash
  cd react-native-flight-tracking
   ```
### 3. Install necessary packages
```bash
   npm install
   ```

### 4. Start the app

```bash
    npx expo start
   ```
